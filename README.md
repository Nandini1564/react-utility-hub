# React Utility Hub 🚀

A simple utility application built using React, Vite, and Tailwind CSS.

## Features

### 🌐 Text Translator

* Translate English text to other languages.
* RapidAPI integration support.
* Clean and responsive UI.

### 🔀 Random String Generator

* Generate random alphanumeric strings.
* Implemented using:

  * useState
  * useEffect
  * useCallback

### 🧭 Client-Side Routing

* Implemented using React Router DOM.
* Pages:

  * Home
  * Translator
  * Random Generator

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* Axios
* RapidAPI

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the project root:

```env
VITE_RAPIDAPI_KEY=YOUR_API_KEY
VITE_RAPIDAPI_HOST=google-api31.p.rapidapi.com
```
